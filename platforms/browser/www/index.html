Before using the Spinball pitching machine please adhere to the following warnnings
  • Calibrate Spinball pitching machine before first use or whenever the Spinball pitching machine has been moved
  • Always throw test pitches to make sure the Spinball pitching machine is throwing strikes before allowing a batter to hit
  • The Spinball pitching machine will throw both balls and strikes. Batters must always wear a helmet when hitting picthes thrown by the Spinball pitching machine
  • For best accuracy when using the Spinball pitching machine do not mix types or quality of balls being thrown by the Spinball pitchign machine
